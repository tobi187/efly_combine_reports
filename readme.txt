### Anleitung ###

NoGo's:
    - Alle Dateien die bearbeitet werden sollen dürfen nicht geöffntet sein
    - Die Dateinamen dürfen keine Strichpunkte enthalten
    - Die Dateien sollen (vlt müssen) die Endung .xlsx haben

Steps:
    1. Alle Dateien schließen die bearbeitet werden sollen
    2. Programm starten
    3. Im Popup (kleines Fenster) zuerst die Dateien wählen, aus denen der Report zusammengestellt werden soll (+ Ok drücken)
    4. Im 2. Popup die Datei wählen, in die es gespeichert werden soll (+OK)
    5. Das Programm läuft ab da im Hintergrund durch und sobald es fertig ist, erscheint ein kleines Popup
    -- bitte solange das Programm läuft die Dateien die bearbeitet werden nicht öffnen --
